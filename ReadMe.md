一种基于色彩聚类的乱针绣铺底绣稿生成方法
===================================

简介：
-----------------------------------
利用颜色聚类和图割优化，对图像进行分割

![Image](/data/hua2.jpg)
![Result](/data/hua2.jpg_result.bmp)
  
接口文件列表：
-----------------------------------
ColorCluster.m:		脚本式接口文件

ColorClusterFunc.m:	函数式接口

threadcolor.txt:	绣线颜色库


用法：
-----------------------------------
修改ColorCluster.m中第9~11行，定义图片所在文件夹、图像文件名、聚类类别数K。
